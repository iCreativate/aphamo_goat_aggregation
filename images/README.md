# Images from Aphamo Boerdery

Use photos from **Aphamo Boerdery's Facebook page** for an authentic, on-brand site.

**Facebook page:** [Aphamo Boerdery](https://www.facebook.com/aphamoboerdery)

## How to add images

1. Log into Facebook and open [Aphamo Boerdery](https://www.facebook.com/aphamoboerdery).
2. Go to **Photos** (or **Posts** with images).
3. Open an image, click the three dots (⋯) → **Download** (or right‑click → Save image).
4. Save files into this folder using the names below so the website finds them.
